Platform features in use
🔒

Cloudflare Tunnel

Origin server connected via encrypted outbound-only tunnel. No open inbound firewall ports required.

🛡️

Zero Trust Access

The /secure path is locked behind Cloudflare Access with Google SSO — only approved identities get in.

⚡

Edge Workers

Identity data, flag serving, and routing logic runs at the Cloudflare edge — not on the origin server.

🗄️

R2 Object Storage

Country flag images stored in a private R2 bucket and served via Worker bindings. Try /flags/SG.

🗃️

D1 Database

Flag metadata indexed in a D1 SQLite database, bound to the Worker. Try /flags-d1/SG.

🧱
